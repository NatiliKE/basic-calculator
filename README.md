# Basic Calculator Program

This is a simple Python program that:
- Asks the user to input two numbers.
- Asks for a mathematical operation (+, -, *, /).
- Performs the operation.
- Displays the result.

## Example Run
Enter the first number: 10
Enter the second number: 5
Enter operation (+, -, *, /): +
10.0 + 5.0 = 15.0
